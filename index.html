Projects

  • 2023
  • AutoActivator
  • Autoactivator is a tool designed to streamline the developer's workflow when working with virtual environments. With Autoactivator, the user can easily activate virtual environments as soon as they enter a corresponding project directory, and stay activated for as long as they're working in that directory.
    Once they're done with the project and move out of the directory, Autoactivator automatically deactivates the virtual environment for the user, ensuring that they don't accidentally use the wrong environment in future projects.
    This feature makes it easier for developers and programmers to manage multiple virtual environments without having to manually activate or deactivate them every time they switch projects.
  • Currently utilized daily by the Quicktext data science team, saving time and streamlining redundant tasks.
